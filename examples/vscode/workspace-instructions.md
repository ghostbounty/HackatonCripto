# Ejemplo para VS Code / workspace instructions

## Instrucciones del workspace

- Este repo usa `AGENTS.md` como índice operativo.
- La carpeta `skills/` define sub-flujos reutilizables.
- Toda propuesta o cambio trazable debe vivir en `changes/<change-name>/`.
- `docs/product/` se considera contexto funcional preservado del dominio.
- `.atl/skill-registry.md` puede regenerarse con `scripts/setup.sh` o `scripts/setup.ps1`.
