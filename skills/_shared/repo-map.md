# Repo Map

## Zonas principales

- `docs/product/`: contexto funcional del dominio del hackathon.
- `docs/`: documentación del framework operativo.
- `changes/`: artifacts obligatorios por change.
- `skills/`: skills operativas para recorrer el flujo.
- `.atl/`: índices regenerables y metadatos auxiliares.
- `scripts/`: bootstrap, setup y validaciones.
- `examples/`: ejemplos de consumo para distintas herramientas.
- `.github/`: templates y workflows de colaboración.
